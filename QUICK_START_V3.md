## 🔧 配置说明

### GitHub Secrets（已配置）

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `JISUAPI_KEY` | 极速数据API密钥 | `your_api_key` |
| `EMAIL_TYPE` | 邮箱类型 | `qq` 或 `163` |
| `EMAIL_ADDRESS` | 发件人邮箱 | `your@qq.com` |
| `APP_PASSWORD` | 邮箱授权码 | `your_password` |
| `RECIPIENT_EMAILS` | 收件人邮箱 | `user@example.com` |

## ❓ 常见问题

### Q1: 如何查看运行日志？

**A**:
1. 访问：https://github.com/lgd3206/jicunjin/actions
2. 点击最新的运行记录
3. 点击 `运行金价监控` 查看详细日志

### Q2: 如何修改下跌阈值？

**A**:
1. 进入 GitHub 仓库
2. Settings → Secrets and variables → Actions
3. 编辑 `DROP_THRESHOLD_PERCENT`
4. 修改为你想要的值（如 `3.0` 表示3%）

### Q3: 如何暂停监控？

**A**:
1. 进入 GitHub 仓库
2. Actions → 积存金价格监控
3. 点击右上角 `...` → Disable workflow

### Q4: 邮件没收到怎么办？

**A**:
1. 检查垃圾邮件文件夹
2. 查看 GitHub Actions 日志确认是否发送成功
3. 检查 `RECIPIENT_EMAILS` 配置是否正确

### Q5: API额度用完了怎么办？

**A**:
- 免费额度：100次/天
- 当前使用：48次/天
- 剩余额度：52次/天
- 如果不够，可以升级极速数据API套餐
