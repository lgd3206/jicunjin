## 立即配置（3步完成）

### 第1步：获取API密钥（2分钟）

1. 访问 **极速数据官网**：https://www.jisuapi.com/
2. 点击右上角"注册/登录"
3. 注册账号（支持手机号/邮箱）
4. 登录后进入"控制台"
5. 找到"黄金数据"API
6. 复制你的 `appkey`（类似：`xxxxxxxxxxxxxxxxxxxxxxxx`）

**免费额度**：100次/天（当前使用48次/天，完全够用）

### 第2步：配置GitHub Secrets（1分钟）

1. 打开你的GitHub仓库：https://github.com/lgd3206/jicunjin
2. 点击 `Settings`（设置）标签
3. 左侧菜单选择 `Secrets and variables` → `Actions`
4. 点击绿色按钮 `New repository secret`
5. 填写信息：
   - **Name（名称）**：`JISUAPI_KEY`
   - **Value（值）**：粘贴你的appkey
6. 点击 `Add secret` 保存

### 第3步：验证功能（1分钟）

1. 进入 `Actions` 标签页
2. 选择左侧的 `积存金价格监控` 工作流
3. 点击右侧 `Run workflow` 下拉菜单
4. 点击绿色按钮 `Run workflow` 确认
5. 等待30秒，刷新页面
6. 点击最新的运行记录查看日志

**成功标志**：
```
✓ 当前金价: XXX.XX 元/克
✓ 正在获取各大银行金价...
✓ 成功获取 X 家银行金价
```

## 🔍 常见问题

### Q1: 如果不配置API密钥会怎样？
**A**: 系统会正常运行，只是邮件中不会显示银行金价对比表格，只显示国际金价。

### Q2: API密钥配置错误怎么办？
**A**:
1. 检查日志中是否有"极速数据API返回异常"
2. 重新检查API密钥是否正确复制
3. 确认API密钥在极速数据官网是否激活

### Q3: 如何查看API使用情况？
**A**: 登录极速数据官网 → 控制台 → 查看"今日调用次数"

### Q4: 运行频率是多少？
**A**: 每30分钟运行一次，每天48次，远低于100次/天的免费额度。

## 🎯 触发提醒的条件

系统会在以下情况发送邮件提醒：

1. **历史最低价**：当前价格是近24小时最低价
2. **大幅下跌**：价格从最高价下跌超过5%（可配置）
3. **价格波动**：24小时价格波动幅度超过2%

## 📱 微信接收提醒

在微信中搜索"**QQ邮箱提醒**"小程序或公众号，绑定你的邮箱账号，即可在微信上实时接收金价提醒。

## 🛠️ 高级配置

### 调整下跌阈值
在GitHub Secrets中修改 `DROP_THRESHOLD_PERCENT`（默认5.0）

### 调整运行频率
修改 `.github/workflows/gold-monitor.yml` 中的cron表达式：
- 每15分钟：`*/15 * * * *`
- 每1小时：`0 * * * *`

### 禁用银行金价
删除GitHub Secrets中的 `JISUAPI_KEY` 即可
